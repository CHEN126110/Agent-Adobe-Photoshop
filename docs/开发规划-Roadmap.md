## 2. 当前阶段: 电商设计自动化 🔧

**目标**: 实现袜子类目详情页和主图的端到端自动化设计

### Phase 5A: 详情页设计 (Week 1) 🎯 **当前优先**

| 任务 | 描述 | 状态 | 预计工时 |
|------|------|------|---------|
| 类型定义 | detail-page.types.ts | 待开发 | 1h |
| 模板解析器 | DetailPageParser | 待开发 | 4h |
| 屏类型推断 | 命名规则 + 关键词匹配 | 待开发 | 2h |
| 图层关系检测 | LayerRelationDetector | 待开发 | 3h |
| 自动修复引擎 | AutoFixer | 待开发 | 3h |
| 内容匹配服务 | ContentMatcher | 待开发 | 3h |
| 批量填充工具 | DetailPageFiller | 待开发 | 3h |
| 切片导出 | SliceExporter | 待开发 | 2h |
| 技能执行器 | detail-page.executor.ts | 待开发 | 3h |

**详情页设计流程**:

```
用户: "帮我做详情页"
     ↓
┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────┐
│ 1. 模板解析  │ → │ 2. 问题检测  │ → │ 3. 智能填充  │ → │ 4. 导出  │
│   结构识别   │    │   自动修复   │    │   验证循环   │    │   切片   │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────┘
     ↓
输出: 详情页切片 (10-15张 JPEG)
```
